## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clonehttps://github.com/GloriaNanana/Snake.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the command
   ```sh
   ng serve
   ```
